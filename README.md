Суть приложения

"Личный планировщик дедлайнов" - это приложение, которое помогает 
студентам (и не только) вовремя выполнять задачи. Пользователь добавляет 
задачи, описание и их дедлайн, а приложение отображает их в удобном 
списке и напоминает о приближении сроков.
Цель приложения - сдавать вовремя задачи.
